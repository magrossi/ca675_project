Face Matcher

This web application will support a user uploading an image of his/her face which is in turn compared against a _big_ dataset of actors/actress faces, resulting in the top ten most similar matches being returned to the user. The system will also give the user the option of adding his/her face to the dataset, making it subject to be matched against other similar-faced users.
